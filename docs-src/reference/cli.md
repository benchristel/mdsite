# Command-Line Interface

## Summary

```bash
# Build your project
mdsite [-i INPUTDIR] [-o OUTPUTDIR] [-t TEMPLATEFILE]

# Generate or update order.txt files
mdsite order [-i INPUTDIR]
```

## Defaults

- `INPUTDIR` defaults to `src`.
- `OUTPUTDIR` defaults to `docs`.
- `TEMPLATEFILE` defaults to `template.html`.
